The Wizard's Spellbook API

Welcome to the ultimate RESTful Harry Potter's spells API

What is this?

This is a full RESTful API linked to an extensive database detailing everything about the official Harry Potter's spells.

Where do I start?

We have awesome documentation on how to use this API. It takes minutes to get started. This API will always be publicly available and will never require any extensive setup process to consume.
