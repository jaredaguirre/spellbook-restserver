No authentication is required to access this API, and all resources are fully open and available.

General Resources

Calling the API endpoint without a resource ID or name will return a paginated list of available resources. This is a view of all the information that you can get from it:

Name Description Type
id This is the unique indentifier used to perform the query searches string
name The well-formatted identifier name that include spaces string
description Contains a brief description of the spell use string
type Indicates the type of spell. There is a total of eight: 'Modifiers', 'Charm', 'Transfiguration', 'Curses','Hexes','Jinxes','Healing Spells'. string

Example GET resource

http://localhost:8080/api/spellbook?id={spell-name}
Example: http://localhost:8080/api/spellbook?id=avada-kedavra
